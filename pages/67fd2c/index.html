      基于 react + ts 开发的 todolist 应用

      这是一个学习 react 过程中产生的个人项目，不定期更新

      # ✨ 特性

      • [x] 任务支持拖拽来改变状态.

      • [x] 创建任务

      • [x] 删除任务

      • [x] 改变任务状态为进行中或者完成

      • [x] 任务跟踪

      • [x] 当一个项目完成后进行状态锁定

      • [x] 任务进度显示

      • [ ] 客户端

      # 🚀 开始

      # 克隆

      git clone https://github.com/stefan-ysh/react-todo.git

      # 安装依赖

      npm install

      # 启动

      在开发模式下运行应用，打开 http://localhost:3000 (opens new window) 在浏览器预览

      npm start

      # 打包

      将用于生产的应用程序生成到build文件夹

      npm run build
