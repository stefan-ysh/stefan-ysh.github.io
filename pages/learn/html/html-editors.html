    • Notepad++ (opens new window)

    • VSCode (opens new window) (Visual Studio Code 个人推荐）
    • WebStorm (opens new window)

    # 使用方法（以 VSCode 为例）

    # 1. 安装 VSCode

    下载地址 (opens new window)，选择你是用系统的版本进行安装，安装步骤都是傻瓜式的，不需要任何设置，这里就不赘述了。

    # 2. 创建 html 文件

    # 3. 在文件中添加代码

    • 快捷键 ! ：输入后回车即可快速创建 html 结构，如果手写也许将结构写全。ps:英文输入下的叹号

    • 整体结构一键创建

    • 写入需要渲染的标签，这里我写入了一个标题和一句话

    # 4. 保存文件

    • 保存时选择 html 格式文件

    # 4. 打开 HTML 文件查看效果

    • 找到文件储存位置，双击即可打开并看到网页效果，bingo！
